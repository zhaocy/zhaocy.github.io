装机必备

发表于 2017-08-30 | 分类于 Mac | 字数统计:1,802 阅读时长预计: 7 min

我的本本是MacBook Pro 2016款 15寸，用了有一段时间了，给大家分享下我在生活和工作中一些常用软件，破解的软件在博客里有独立介绍破解方法。

社交类

  • 钉钉

    办公协作软件，工作需要，每天都在使用。桌面、手机都有跨平台版本。App Store下载

    🖥 windows Mac iphone Android

  • 微信

    国民软件，生活工作必不可少。App Store下载

    🖥 windows Mac iphone Android

  • QQ

    社区娱乐聊天软件，曾经的聊天神器，现在很少使用了，几乎成为鸡肋。 App Store下载

    🖥 windows Mac iphone Android

  • 阿里旺旺

    淘宝、天猫购物必备。 App Store下载

    🖥 windows Mac iphone Android

  • Mapio

    新浪微博客户端工具。 App Store下载。

    🖥 windows Mac iphone Android

上传下载

  • 迅雷

    下载利器，曾经的流氓软件，App Store下载。

    🖥 windows Mac iphone Android

  • Yummy FTP

    FTP客户端工具，开发运维使用，博客分享了破解版。

    🖥 Mac

远程连接

  • EasyConnect

    连接阿里云的VPN工具，工作之需。官网提供。

🖥 `Windows` `Mac`

  • vSSH

    SSH客户端工具，做开发的都懂。博客分享了破解版。

    🖥 Mac

百度网盘

做为众多垂死挣扎的免费云盘的最终幸存者，从刚上线就在用，里面已经放了很多学习资料。App Store下载。

——- 华丽的分割线1 ——-

IntelliJ IDEA

从Eclipse痛苦的转移到幸福使用的神器，集成开发工具必备。博客分享了破解版。

Chrome

从网页浏览器本身功能上看Safari已经非常好用了，但是作为开发者来看Chrome还是首选。 App Store下载。

Postman

Http请求客户端工具，开发者必备。Chrome商店下载。

Medis

Redis客户端，比RedisDesktopManager优点是有CLI,缺点是key没有树形界面。 App Store下载，我是花了30RMB的。

Docker

容器管理服务器，开发者都懂不解释了，官方下载。

SourceTree

Git本地和远程仓库管理工具，官方下载。

DataGrip

多引擎数据库管理工具，IntelliJ 系列产品，强大的代码提示和补全功能，博客分享了破解版。

OmniGraffle

绘图工具，流程图、时序图、类图、UI原型图、思维导图...无所不能，正版非常规，博客分享了破解版。

Beyond Compare

文本文件比较工具，开发人员常用，不过很多IDEA工具都集成了。博客分享了破解版。

Microsoft office

这个不用解释了，地球人都知道的办公必备软件。博客分享了破解版。

——- 华丽的分割线2 ——-

1Password

高强度加密你的登录账号、服务器密码、数据库密码…,尽管Mac自带密钥工具，但是1Password是多终端的。博客分享了破解版。

有道云笔记

记录个人学习、工作、生活笔记，支持MD,密码箱，诸如此类笔记本还有很多，笔记印象笔记、蚂蚁笔记等，各有特色。App Store有下载。

Teambition

团队项目协作任务管理软件，多终端的，经常用。AppStore可下载。

EdrawMax

中文名称叫亿图，顾名思义，可以绘制亿张图，windows和mac下都用过绘制各种图，但遇到Omni Graffle，还是倾向后者。博客分享了破解版。

Dash

各种API文档管理工具，还可以添加笔记，开发者必备，Windows平台下有个类似的工具 - Velocity. 博客分享了Dash破解版。

Near Lock

一款可以通过Iphone锁屏Mac的软件，还可以拍照记录解锁时Mac前的场景。AppStore下载。

Gemini

可以找出所有相同的文件，一遍节约存储空间。博客分享了破解版。

CleanMyMac

清理和优化Mac内存，还有卸载软件等功能，装备必备啊。博客分享了破解版。

Camtasia

一款非常专业的录屏软件，强大的视频剪辑功能，用过的都知道。博客分享了破解版。

Sublime Text

文本编辑工具，众多插件支持，编写代码都非常快捷。博客分享了破解版。

——- 华丽的分割线3 ——-

Star UML

UML绘图工具，应该是Mac下一款最好用的了，韩国人开发的。博客分享了破解版。

XMind

思维导图，比起Windows下的版本，功能上阉割了不少，Mindjet Manager在Mac下又非常卡，OmniGraffle又有点力不从心。博客分享了破解版。

BetterZip

压缩软件，支持常用的压缩格式，装备必备。博客分享了破解版。

calibre

Kindle电子书制作必备软件，博客分享下载。

Paste

粘贴板管理软件，每次复制的项目不再被上次复制的覆盖，可以有选择的进行粘贴和管理。博客分享了破解版。

Magnet

窗口管理软件，可以通过快捷键快速的将桌面窗口已任意姿势摆放。博客分享了破解版。

PDF Expert

顾名思义，这是一款PDF专家，可以编辑、加密、签名等你的PDF。博客分享了破解版。

NewFileMenu

回顾windows资源管理器的右键和工具栏菜单真是丰富啊，可惜Mac下没有这个设计，不过NewFileMenu可以帮你找回Windows操作习惯了。博客分享了破解版。

网易有道词典

阅读英文资料，撰写英文邮件不可或缺的软件，手机APP还有拍照翻译功能，太强大了。AppStore可以下载。

Charles

网络抓包工具，你可能不常用。博客分享了破解版。

——- 华丽的分割线4 ——-

Path Finder

文件管理器，极大方便文件复制、剪切、隐藏等操作。博客分享了破解版。

Picase

老牌子的图片管理软件。AppStore可下载。

百度云同步盘

不想手动上传文件至百度云盘，那么使用同步盘吧。 AppStore可下载。

PhoneClean

iphone管理软件，跟普通的手机管理软件类似。博客分享了破解版。

Parallels Desktop

虚拟机软件，功能异常强大，它可以让你无感知的在不同的系统之间操作。通俗的讲你操作虚拟机下的系统就好像在主系统下操作一样。博客分享了破解版。

DEVONthink Pro Office

文件管理软件，通过这款软件不仅可以搜索文件名，甚至文件的内容也可以检索的到，这是我使用它的理由，充分吧。类似的工具有Together。博客分享了破解版。

Alfred

Spotlight搜索的替代品，因为它足够强大，博客分享了破解版。

Polarr Photo Editor

照片编辑器，类似美图秀秀，我不常用，但感觉需要装一个。博客分享了破解版。

duet

全名是Duet Display,多屏/屏幕扩展工具软件，可以将屏幕扩展到手机和平板电脑。AppStore可下载，收费的哦。

Navicat Premium

多引擎的数据库管理软件，新版的增加了云项目。博客分享了破解版。

——- 华丽的分割线5 ——-

Snagit

强大的截图软件，多种模式的截图，windows下一直用。博客分享了破解版。
