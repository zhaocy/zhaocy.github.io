Duet Display

发表于 2017-08-29 | 分类于 Mac | 字数统计:209 阅读时长预计: 1 min

介绍
Duet Display 的应用可以将 iOS 设备变成 Mac 的第二块屏幕，这样你就可以在尽可能较低的成本下，扩展 Mac 的屏幕面积，从而大大的提高工作效率。

使用方法
通过 USB 连接线将 iPad 扩展成 Mac / PC 的外接显示屏幕，或同时通过VGA或HDMI扩展。

同款比较
TwomonUSB 的应用同Duet Display功能类似，区别：

  • 主机端： Duet Display 除了 OS X 10.9 以上之外，Windows 系统必须是 Windows 7 以上，而 TwomonUSB 最低系统需求是 Windows XP；
  • 扩展端：Duet Display 仅支持 iPad 作为扩展屏幕（最低 iPad 2 起），而 TwomonUSB 即支持 iPad（官方甚至声称支持 iPad 第一代），还支持 Android 平板。
