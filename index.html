移动硬盘安装windows To Go的体验

发表于 2018-03-15 | 分类于 Mac | 字数统计:667 阅读时长预计: 2 min

Win to go是Mac和Win双系统的解决办法之一，或许不是最好，但也不赖！

–

WTG是什么东东，可以用来干什么

windows To Go 简称WTG，就是把Windows镜像装在USB设备里，制作引导启动，将Win系统装在USB设备里。USB设备可以是U盘、移动硬盘、SSD+SSD3.0硬盘盒。

  • 优点：解放了Macintosh HD硬盘空间，留作安装装Mac OSX 系统下的文件。

  • 缺点：
    1.需要随身携带USB设备；
    2.本人希捷 USB3.0 移动硬盘在启动win10系统时，大致耗时半分钟（较慢），但是运行起来无卡顿现象。

方案实现

（本文示例使用的是移动硬盘USB3.0）

准备工作：

  • 硬盘硬盘、U盘、SSD
  • windows10的镜像文件这里可以下载
  • 有线的键盘、鼠标（安装windows时要用到）

  1. 在windows系统下做引导盘

选项1.选择你的USB（作为系统盘）的设备，如U盘，移动硬盘。移动设备一定要把资料先备(teng)份(kong)，因为WTG会格式化你的USB设备。针对移动硬盘设备要强调的是无论此移动硬盘分区不分区都会被格式化，所以分区了的移动硬盘一定要备份好资料！备份好资料！备份好资料！
选项2.下载好的iso格式Win10镜像（推荐去MSDN下载，因为本人用自己台式机镜像出来的Win10.iso失败了）解压，点击“浏览”选择Win10镜像sources文件夹下的install.wim文件；
选项3.勾选VHD格式选项4.勾选UEFI+GPT

点击“创建” 耐心等待十几分钟到二十分钟的样子（速度取决你的设备），完成后会弹出窗口，提示创建成功！

注意：
a). 360安全卫士一定要关闭
b). 建议不要在虚机上操作

2.在Mac 上，打开启动转换助理.app，点击菜单->操作->下载windows支持软件，这里是驱动驱动程序，备用，下一步安装好windows ，拷贝到windows系统安装即可。

3.将移动硬盘插入Mac电脑的USB接口上，点击 开机键，同时按住option健，选择EFI Boot，回车。然后 Win10准备界面→设备准备→重启→输入电脑用户名→设置开机→Cortana→“诗词名言”→登录到Win10界面了。至此，Win10基本安装成功。

接下来就是驱动：进入用之前BootCamp下载好的Win10驱动程序文件，双击Setup后，走完安装程序后，Win10就安装成功了。

提高效率的Mac技巧

发表于 2017-11-16 | 分类于 Mac | 字数统计:1,095 阅读时长预计: 4 min

系统原生

系统自带的功能、快捷键等。

快捷键功能篇

强制退出程序

Option-Command-Esc

当你遇到整个 Finder 卡死，Dock 栏右键都点不开，系统一直转圈圈，根本没办法强制重启 Finder 的时候：按 Option-Command-Esc 来打开强制退出应用程序，亲测有效。
几种死机时重启 Finder 方法的强力程度：
Option-右键 强制重启 < 用 强制退出应用程序重启 < 用 Terminal 关闭 < 长按关机键强制关机。

向后退格

Fn + Delete

在 Windows 下的 Del 键可以实现『向后退格』（删除光标后面的字符），Mac 下怎么实现呢？
用 Fn-Delete 就可以啦。

快速调到地址栏

command + L

浏览网页的时候，怎么快速跳到地址栏？
Command-L，功能和火狐的 F6 类似，聚焦到地址栏，直接修改内容。应该是常见快捷键吧，不过发现的时候高兴了好一会儿。

快速跳到桌面

command + D

在保存文件的时候，按 Command-D 可以快速跳到桌面。
其他几个默认文件夹同理，快捷键都显示在 『菜单-前往』 里啦。

选中多个文件放至新文件夹

control + command + N

选中多个文件，按 Control-Command-N 可以直接把它们放到一个新建文件夹中。

emoji表情

control + command + space

速输Apple logo

option+shift+K

在任意输入框中，按option+shift+K组合键。
