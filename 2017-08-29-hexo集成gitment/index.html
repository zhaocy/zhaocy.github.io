hexo集成gitment

发表于 2017-08-29 | 分类于 技术 | 字数统计:705 阅读时长预计: 4 min

gitment集成到hexo网上也有一些文章，但是都不是很准确，支离破碎的，对此我梳理了一番。

步骤一：注册OAuth App

注册OAuth App前提你要有gitHub账号，打开连接 注册一个OAuth Application 。

  • Application name 自己随便想一个
  • Homepage URL 填写一个完整的URL，比如是你的博客的域名。
  • Application description 描述（不是必填项）
  • Authorization callback URL 是回调地址，主要是为了在你的博客所集成的gitment中，单击Login验证，然后再跳转你的博客域名（即回调地址）。

一切就绪，”Register application” ，成功后会返回Client ID 和 Client Secret
后面在配置gitment时会用到。
