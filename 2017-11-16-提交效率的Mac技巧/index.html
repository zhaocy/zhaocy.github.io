提交效率的Mac技巧

[TOC]

##系统原生

系统自带的功能、快捷键等。

###快捷键功能篇

####强制退出程序

Option-Command-Esc

当你遇到整个 Finder 卡死，Dock 栏右键都点不开，系统一直转圈圈，根本没办法强制重启 Finder 的时候：按 Option-Command-Esc 来打开强制退出应用程序，亲测有效。
几种死机时重启 Finder 方法的强力程度：
Option-右键 强制重启 < 用 强制退出应用程序重启 < 用 Terminal 关闭 < 长按关机键强制关机。

####向后退格

Fn + Delete

在 Windows 下的 Del 键可以实现『向后退格』（删除光标后面的字符），Mac 下怎么实现呢？
用 Fn-Delete 就可以啦。

####快速调到地址栏

command + L

浏览网页的时候，怎么快速跳到地址栏？
Command-L，功能和火狐的 F6 类似，聚焦到地址栏，直接修改内容。应该是常见快捷键吧，不过发现的时候高兴了好一会儿。

####快速跳到桌面

command + D

在保存文件的时候，按 Command-D 可以快速跳到桌面。
其他几个默认文件夹同理，快捷键都显示在 『菜单-前往』 里啦。

####选中多个文件放至新文件夹

control + command + N

选中多个文件，按 Control-Command-N 可以直接把它们放到一个新建文件夹中。

####emoji表情

control + command + space

####速输Apple logo

option+shift+K

在任意输入框中，按option+shift+K组合键。

####微调音量/亮度

option + shift

按住 Option-Shift 可以以四分之一格为单位调节音量/亮度。

####查看简介

command +i

如果想查看文件或文件夹或者应用程序的简介，选中该文件或应用程序，按 command+i 组合键即可

####切换应用程序

command + shift + tab

按住command键再按tab键，可从左往右切换已打开的应用程序；command+shift+tab，可从右往左切换。

####分屏功能

方法一：当窗口不是出于全屏状态时，长按两秒左上角的绿色按钮松手。
方法二：在触控板上四指上推，将下方的小窗口拖入上方的全屏状态的窗口即可。

####外键键盘按键映射

用的 外置键盘 是传统 Windows 键位，使用 Mac 的快捷键很不方便怎么办？
修改修饰键，将按键映射成习惯的功能。比如我将外接键盘的 Ctrl 映射成了 Command ，这样做『保存（Ctrl+S）』之类的就方便很多，更符合习惯。

####快速查单词

OS X提供了三指轻拍查找的功能，把光标移到单词或汉字上面，无需选中，三指轻拍触控板，系统词典就会显示词语释义，该功能可以在系统偏好设置的触控板里设置。

####速输域名

在Safari浏览器中访问.com的域名，如baidu.com直接输入baidu/回车即可，也就是在域名后面加上“/”再回车即可快速访问。

###剪切功能

怀念 Windows 的 Ctrl-X 剪切文件？
Mac 下用 Command-C 配合 Option-Command-V 可以实现这个功能，但是如果想要直接得 Command-X 剪切文件的话，可以安装 XtraFinder。
